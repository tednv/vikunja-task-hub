# Branding

Vikunja Task Hub uses a dark navy, violet, and cyan visual identity built around a friendly llama, a progress ring, and a task check mark.

## Repository assets

- `assets/vikunja-task-hub-banner.png` is the primary repository and documentation hero.
- `assets/vikunja-task-hub-icon.png` is the square project and integration icon source.

Keep the icon square and preserve clear space around the mascot and progress ring. Do not add third-party logos to either asset or use the artwork in a way that implies endorsement by Vikunja or Home Assistant.

## Palette

| Role | Color |
| --- | --- |
| Midnight background | `#050A28` |
| Deep navy | `#0D1B50` |
| Indigo | `#2537C7` |
| Violet | `#8B3DFF` |
| Cyan | `#24A9FF` |
| White | `#FFFFFF` |
| Muted text | `#A9B6D5` |

The values are a practical UI palette derived from the supplied brand direction. They are not color samples guaranteed to match every pixel in the raster artwork.

## Naming

Use **Vikunja Task Hub** on first reference. The compatibility identifiers remain `vikunja` for the Home Assistant integration domain and `custom:vikunja-todo-card` for the dashboard card type.
